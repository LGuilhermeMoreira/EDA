Para executar o programa, basta digitar o comando "make". 
Esse comando aciona o Makefile, 
que por sua vez executa todos os comandos 
necessários para compilar o programa.

A função search foi feita na propria arvore e atraves de uma flag, é deifinida
o tipo de busca

sendo 3 tipos de busca

1. Por CPF
2. Por nome
3. Por uma data específica
4. Por um intervalo de datas

Após a pesquisa de busca, caso a saida seja 
"=======================================", é
que não foi encontrado nenhuma pessoa.
